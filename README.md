## 🪑 boardroom

[![boardroom](https://raw.githubusercontent.com/Tenerit/boardroom/master/assets/social-preview.png)](https://github.com/Tenerit/boardroom)

A whole-project **review board** for Claude Code. A panel of expert hats —
architect, security, SRE, UX, product, investor, skeptic, cost — reviews your
project in parallel and hands you a **GO/NO-GO decision** plus the cross-discipline
trade-offs only you can settle. Read-only, MIT.

```
/plugin marketplace add Tenerit/boardroom
/plugin install boardroom@tenerit
```

---

## 🩻 ccx — Claude Code X-Ray

[![ccx](https://raw.githubusercontent.com/Tenerit/ccx/master/assets/social-preview.png)](https://github.com/Tenerit/ccx)

Cross-session **cost & token-waste analytics** from the logs Claude Code already
writes. See what your sessions actually cost and what quietly burned tokens —
duplicate reads, fat tool dumps, cache churn. Zero network, zero deps, MIT.

```
npx @tenerit/ccx cost
```
